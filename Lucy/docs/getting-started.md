**If Else Statement**
```bash
c = 20;
d = 30;
if(c > d){
  show("c :" + c + "is bigger than d : " + d);
}
else{
  show("else");
}
```

**While statement**

```bash
i = 2;
while i < 10
{
  show(i);
	i = i + 2;
}
```
